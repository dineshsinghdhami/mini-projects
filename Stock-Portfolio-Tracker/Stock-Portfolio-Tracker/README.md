# Stock Portfolio Tracker 📊

[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)]()

A simple **command-line Stock Portfolio Tracker** built using Python 🐍 to calculate and save investment values.

---

## 🌟 Features

- 📈 Track multiple stocks in a portfolio
- 🏷️ Predefined stock prices (AAPL, TSLA, GOOGL, AMZN)
- 🔢 Enter quantity of shares owned
- 💰 Automatically calculates individual & total investment value
- 💾 Option to save portfolio details to a text file
- ⚠️ Validates stock availability

---

## 🕹️ How It Works

1. Run the program.
2. Enter how many different stocks you own.
3. Input stock symbols (e.g., AAPL, TSLA).
4. Enter the number of shares for each stock.
5. The program calculates:
   - Individual stock value
   - Total investment value
6. Choose whether to save the portfolio to `portfolio.txt`.
