# Hangman Game 🎯

[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)]()

A simple **command-line Hangman game** built using Python 🐍.

---

## 🌟 Features

- 🎮 Classic Hangman gameplay in the terminal
- 🔤 Guess letters one at a time
- ❌ Limited attempts (6 chances)
- 🧠 Random word selection from a predefined list
- 📢 Clear messages for correct & incorrect guesses
- 🏆 Win/Lose detection with final word reveal

---

## 🕹️ How to Play

1. Run the program.
2. A random word is selected.
3. Guess one letter at a time.
4. Correct guesses reveal letters in the word.
5. Incorrect guesses reduce remaining attempts.
6. Guess the full word before attempts reach zero to win!

## 💻 Technologies Used

- Python 🐍
- Built-in `random` module
